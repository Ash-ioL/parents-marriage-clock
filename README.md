# My Parents' Marriage Clock

<hr>

A marriage clock tracking the years, days, minutes, and seconds that my parents have been married to each other.

<hr> 

Happy wedding anniversary mom and dad! Have a wonderful --th anniversary and I hope you like this little gift I made.

<hr>

## Features

- Heart visual effects
- Display of the time my parents have been married in seconds, minutes, hours, days, months, and years.
- Responsive and aesthetic feel
- A kind letter to my parents
- Smooth swaps between the clock page and the letter page through arrow button navigation

## How to use it

- Click the deployed link
- Explore the page through seamless arrow navigation
- Sit back and watch the magic
